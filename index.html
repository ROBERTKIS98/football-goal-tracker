Google Sheets Integration

This dashboard pulls data directly from a Google Sheet. Update the stats in the sheet and see them reflected here automatically.

How to set up:

  1. Make a copy of the template Google Sheet
  2. Publish it to the web (File > Share > Publish to web)
  3. Copy the Sheet ID and update the JavaScript code
  4. Start updating player stats in the sheet
